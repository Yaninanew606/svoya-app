## Анкета (11 шагов)

1. **Возраст** — слайдер 35-70
2. **Цель** — до 2: тонус, снижение веса, энергия, осанка
3. **Уровень подготовки** — новичок / возвращаюсь / занимаюсь / активно
4. **Время на тренировку** — 15 / 20 / 30 / 45 мин
5. **Предпочтения в питании** — до 3: без мяса (`no_meat`), без молочного (`no_dairy`), без глютена (`no_gluten`), мин. сахара (`low_sugar`), интервальное голодание (`intermittent_fasting`)
6. **Режим дня** — ранний / стандартный / поздний / ненормированный
7. **Тип тренировок** — до 3: силовые, кардио, растяжка, йога, ортопедическая гимнастика
8. **Ограничения по здоровью** — колени, спина, давление, варикоз, шея, другое
9. **Замеры** (необязательно) — рост, вес, талия
10. **Цикл и гормоны** — регулярный / нерегулярный / перименопауза / менопауза + дата последней менструации
11. **Особенности** — роды, сидячая работа, головные боли, отёки, приливы, проблемы со сном

## Персональный онбординг (после генерации)

**Слайд 1 — "Вот что я для тебя подобрала":**
- Имя из Telegram + возраст
- Динамические карточки на основе ответов анкеты:
  - Цикл: "Я учла фазы твоего цикла" / "Программа адаптирована под перименопаузу"
  - Еда: "Все блюда без мяса" / "Питание в окне 8 часов"
  - Здоровье: "Добавлены упражнения для тазового дна" / "Акцент на осанку"
  - Уровень: "Начинаем мягко — нагрузку увеличим постепенно"
  - Ограничения: "Исключены прыжки — берегём колени"
- Если указана дата цикла: карточка фазы с цветной индикацией

**Слайд 2 — "План на неделю":**
- Мини-расписание (7 цветных блоков)
- Статистика: X силовых, Y кардио, Z растяжка, W отдых
- Калории в день, время тренировки
- "Начнём!" → /nutrition, "Изменить расписание" → /workout (edit mode)

## Питание

### Главный экран (/nutrition)
- **Приветствие**: "Привет, {имя}" + случайная мотивация (14 вариантов)
- **Бейдж цикла**: "День X цикла · Фаза · Интенсивность"
- **Неделя**: полоска Пн-Вс с переключением
- **Кнопки-пилюли**: [ИГ] [Вода] — нажатие открывает выдвижную панель справа
- **Макросы**: калории / Б / Ж / У
- **Карточки приёмов пищи** с временем из настроек ИГ

### Интервальное голодание
- Кнопка "ИГ" — всегда видна, серая (выкл) / зелёная (вкл)
- Нажатие → выдвижная панель справа:
  - Таймлайн 24ч: зелёная зона = окно приёма пищи
  - Точки = время приёмов, вертикальная линия = сейчас
  - Статус: "Окно приёма пищи до XX:00" / "Голодание"
  - Настройка: выбор часов начала/конца
  - Пресеты: 16:8 рано, 16:8, 18:6, 14:10
  - Кнопка "Выключить"
- **Связь с рецептами**: время в карточках рецептов рассчитывается из настроек окна
- **Логика приёмов**:
  - Окно 8+ часов: 3 полноценных приёма (Первый приём, Ужин, Третий приём), без перекусов
  - Окно ≤6 часов: 2 приёма (Первый приём, Ужин)
  - Без ИГ: Завтрак, Обед, Перекус, Ужин
  - Завтрак и перекус скрываются при ИГ

### Трекер воды
- Кнопка "Вода" → выдвижная панель
- Круговое кольцо: 0/8 стаканов, кнопки +/−
- Сбрасывается ежедневно, localStorage

### Список продуктов на неделю
- Чеклист с чекбоксами
- Копирование и скачивание (.txt)

### Дневник питания (/diary)
- Кнопка "Дневник питания" → /diary
- План vs Факт: таблица с цветовой разметкой
- Таймлайн окна голодания
- Логирование: текст + фото → AI-оценка калорий (keyword-based MVP)
- Итог дня: прогресс-бар + AI-подсказка

## Тренировки

### Экран (/workout)
- **Неделя**: полоска Пн-Вс (всегда первая)
- **Бейдж цикла**: карточка с рекомендацией по фазе
- **Настройка дней**: кнопка "Настроить" → нажми на цветной тег типа тренировки чтобы переключить (Силовая → Кардио → Растяжка → Отдых → ...)
- **Кнопка "Начать тренировку"** — сразу видна, до списка упражнений
- **Секции**: Разминка, Основной блок, Заминка (свёрнутые)

### Недельная ротация
- 2-3 дня силовые (приседания, планка, ягодичный мост, Бубновский)
- 1-2 дня кардио (марш, степ, скалолаз)
- 1-2 дня растяжка/гибкость (йога, мобильность)
- 1+ день отдых (с активным восстановлением)

### Цикло-адаптивные тренировки
| Фаза | Дни | Интенсивность | Рекомендации | Избегать |
|------|-----|---------------|-------------|----------|
| Менструальная | 1-5 | Лёгкая | Йога, растяжка, подъём таза | Прыжки, тяжёлые силовые |
| Фолликулярная | 6-13 | Высокая | Силовые, HIIT — пик энергии | Нет ограничений |
| Овуляция | 14-16 | Средняя | Умеренно, акцент на технику | Взрывные движения (риск связок) |
| Лютеиновая | 17-28 | Средняя | Кардио, йога, растяжка | Максимальные нагрузки |

### Пошаговый режим
- Одно упражнение за раз, таймер обратного отсчёта
- "Готово" / "Пропустить", прогресс-бар
- По завершении → экран "Отличная работа" → чек-ин

### Умные дни отдыха
- Мягкая йога (10 мин, 5 упражнений)
- Мобильность суставов (8 мин, 5 упражнений)
- Дыхательная практика (5 мин, 3 упражнения)
- Кросс-промо BreathFlow

### Тазовое дно (/pelvic-floor)
- 4 упражнения Кегеля с визуальным пульсирующим кругом
- Медленное сжатие, быстрые сокращения, удержание, лифт
- Таймер + хаптик-фидбек
- Карточка в WorkoutScreen: "Тазовое дно · 3 мин"

## Чек-ин (5 шагов + итог)

1. Самочувствие (1-4, цветовые карточки)
2. Сон (да / нет / почти)
3. Питание (да / частично / нет)
4. Тренировка (да / частично / нет / пропустила)
5. Симптомы (приливы, головная боль, суставы, отёки, вздутие, тревожность, бессонница)
6. Итоги дня + 3 кнопки (всё хорошо / тяжёлый день / нужен легче план)

## Backend API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/generate-plan` | Генерация плана (questionnaire + preferences), возвращает cyclePhase + firstName |
| GET | `/api/plan/today` | Текущий план из User.currentPlan (с weeklyWorkout, cyclePhase, firstName) |
| POST | `/api/checkin` | Чек-ин + адаптивная логика + мотивация |
| POST | `/api/motivation` | Мотивационное сообщение |

### Мок-данные (без API ключа)
- Учитывают пищевые предпочтения (вегетарианство, без глютена, без молочного)
- Интервальное голодание (убирает завтрак)
- Недельная ротация по уровню подготовки
- Цикло-адаптивные тренировки (корректировка по фазе)
- Калории по формуле Миффлина-Сан Жеора
